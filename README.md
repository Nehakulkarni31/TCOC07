# Campus Notifications Microservice

## Features

- Protected API Integration
- Notification Priority System
- Top 10 Important Notifications
- Logging Middleware
- Express.js Backend

## Tech Stack

- Node.js
- Express.js
- Axios

## Run Project

cd backend

npm install

node server.js

## API Endpoint

GET /notifications

## Priority Logic

Placement > Result > Event

Latest notifications are prioritized within the same category.
